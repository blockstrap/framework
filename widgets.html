The Default HTML5 Web Wallet

  • KEY GENERATION

    (click button below to open a key-generator)

    Simply add the bs-generate class to any element on pages with the blockstrap widgets module activated

    READ DOCUMENTATION

    Or embed by copy and pasting the following default code directly into the HTML where you would like the button to appear for it to be automatically generated

    <script

  src="/blockstrap/js/widget.js" 
  data-type="btn-generate"
  data-chain="btc"
  data-salt="ADD_SALT_HERE"
    
></script>

  • DONATIONS

    (click button to open a donation window)

    Simply add the bs-donate class to any element on pages with the blockstrap widgets module activated

></script>
